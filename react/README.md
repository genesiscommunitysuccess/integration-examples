## Installation

To generate new genesis project with support for react execture command below in terminal

```bash
npx -y @genesislcap/genx@latest init genesis-poc --no-npm -x  --no-shell --framework React
```

Navigate to client directory, install dependencies and run development server
```bash
cd genesis-poc/client && npm run bootstrap && npm run dev
```
